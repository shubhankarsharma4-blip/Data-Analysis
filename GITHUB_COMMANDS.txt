## ðŸ†˜ Troubleshooting

### Error: "fatal: origin already exists"
```powershell
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/ecommerce-etl-project.git
git push -u origin main
```

### Error: "Permission denied"
Make sure:
- GitHub account created
- SSH key configured or using HTTPS
- Correct URL copied

### Error: "fatal: No commits yet"
```powershell
git add .
git commit -m "Initial commit"
git push -u origin main
```

### Error: "Merge conflicts"
```powershell
git stash
git pull origin main
git stash pop
```

## ðŸ’¡ Pro Tips

1. **Setup local config once**:
   ```powershell
   git config --global user.name "Your Name"
   git config --global user.email "your.email@github.com"
   ```

2. **Add SSH key** (more secure):
   - Generate: `ssh-keygen -t ed25519`
   - Add to GitHub Settings
   - Use: `git@github.com:USERNAME/repo.git`

3. **Create `.gitconfig`** (optional):
   ```
   [user]
       name = Your Name
       email = your.email@github.com
   [core]
       editor = code
   ```
